# Three Rogue Round

Browser 2D game scaffold (match‑3 + turn-based roguelike combat).

## Quick start

Prereqs: Node.js + pnpm.

```bash
pnpm install
pnpm dev
```

## Quality gates

```bash
pnpm lint
pnpm typecheck
pnpm test
pnpm docs:lint
```

Docs live in `docs/` (system of record). Start at `docs/INDEX.md`.
